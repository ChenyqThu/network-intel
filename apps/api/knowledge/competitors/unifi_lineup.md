# UniFi (Ubiquiti) 产品线参考

供 RAG `background` 检索的竞品基础事实。用于写更准确的 impact_note / 威胁研判，不要逐字复述。

## 网关 / 一体机 (Gateways)
- **UniFi Express / Express 7**：入门一体化网关 + AP，直接对标 Omada 入门线。固件随 UniFi OS 迭代（如 5.1.x RC）。
- **Cloud Gateway / Dream Machine 系列**：中高端网关，含 10G、PON 选项，切入运营商/企业场景。

## 接入点 (Access Points)
- **U6 系列**：Wi-Fi 6 主力。**U7 系列**（U7 Pro / Pro Max / Pro XG）：Wi-Fi 7。社区反复出现 U7 Pro XG 高端 AP "覆盖一致性"不及定价的吐槽——Omada 高端 AP 的差异化切入点。

## 蜂窝 / 备份
- **UniFi 5G Backup**：即插式 5G WAN failover，补齐 WAN 冗余短板。社区痛点：缺少手动运营商选择（manual carrier select）。

## 生态护城河
- UniFi 的 KOL/社区内容生态（YouTube 教程、r/Ubiquiti）是品牌护城河；高播放教程反映其内容营销势能。

## 对 Omada 的含义（判断框架）
- UniFi 出新品/固件提速 → **threat**：关注其入门线功能补齐速度。
- UniFi 暴露体验痛点（选网、覆盖、定价） → **opportunity**：Omada 可做差异化卖点。
